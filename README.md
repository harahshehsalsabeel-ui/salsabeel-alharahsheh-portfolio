# Personal Portfolio Website

This is my personal portfolio website, created to showcase who I am, my background, and my journey as a Computer Science student.

## About the Project
The website presents my personal information, education, skills, hobbies, and contact links in a modern, responsive layout. It also includes animations and interactive elements to reflect my interest in front-end development and design.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design for different screen sizes
- Smooth scrolling and animations
- Interactive navigation menu
- Skills section with progress indicators
- Social media and contact links

## Purpose
This project represents my first step toward becoming a professional Front-End Developer. It documents my learning progress and my ability to build structured and visually engaging web pages.
